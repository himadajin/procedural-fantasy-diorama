# plans/ — 歴史的記録・凍結

本ディレクトリは、実装計画・検証の記録を保存する場所である。完了済みの計画は凍結され、
書き換えない。進行中と明記された計画のみ、進捗ステータスと逸脱の記録の追記・更新を行う。
当時の意思決定と検証結果を、その時点の記録として保存するための場所である。

現行の正は以下を参照する。

- 開発の進め方(ワークフロー・レビュー体制・モデル選択): `../workflow.md`
- 仕様(何であるか・どう見えるか・どう作るか): `../specs/`
- データ契約(WorldModel スキーマ・パイプライン契約): `../contracts/`
