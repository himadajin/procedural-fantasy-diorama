# Procedural Fantasy Diorama 利用ガイド

剣と魔法の異世界を思わせるファンタジー建築の箱庭を、
seed と 6 つのパラメータから自動生成するブラウザアプリです。
生成された箱庭を orbit カメラで眺め、seed とパラメータで生成傾向を変えて遊びます。

## 起動方法

```sh
npm install
npm run dev      # 開発サーバー(表示された URL をブラウザで開く)
```

配布用ビルドは `npm run build`(出力は `dist/`)、
その確認は `npm run preview` で行えます。

初回起動時は設定画面を挟まず、デフォルト seed で自動生成が始まります。
生成中も画面はそのままで、完了すると箱庭がフェードインします。

## カメラ操作

| 操作 | マウス | タッチ |
|---|---|---|
| 回転 | 左ドラッグ | 1 本指ドラッグ |
| ズーム | ホイール | ピンチ |
| パン(平行移動) | 右ドラッグ または Shift+ドラッグ | 2 本指ドラッグ |

画面左下の「⌂」ボタン(Reset Camera)でいつでも初期構図に戻れます。
箱庭を見失ったときに使ってください。

## seed と Generate

- **seed** — 入力欄で自由に表示・編集できます。同じ seed と同じパラメータからは、
  必ず同じ箱庭が再現されます。
- **Randomize Seed**(サイコロボタン)— seed 入力欄を新しい値にするだけで、
  再生成はしません。
- **Generate** — このボタンを押したときだけ、現在の seed とスライダーの値で
  箱庭を再生成します。スライダーを動かしただけでは何も変わりません。

スマホではパネルがボトムシートになります。半開では seed・Randomize・Generate
だけが見え、つまみを引き上げるとスライダーとサマリーが現れます。

## 6 つのパラメータ

すべて 0〜100 のスライダーで、既定値は 50 です。
村や魔法都市などの「モード」を選ぶ仕組みはなく、
6 つの傾向の組み合わせと seed から箱庭の性格が連続的に決まります。

- **World Scale / 敷地規模** — 箱庭全体の広さ、生成される建築量、
  道や外縁の余裕に影響する。
- **Settlement Pressure / 居住圧力** — 建物や道がどれだけ集まりやすいか、
  中心部の密度、外縁への広がりに影響する。
- **Prosperity / 繁栄度** — 建物の整備度、素材の上質さ、
  屋根・窓・梁・道・広場の見栄えに影響する。
- **Water Presence / 水辺の強さ** — 川、池、湖状の水域、街の中を通る水路、
  橋、岸辺、水辺建築に影響する。
- **Arcana / 魔法の濃さ** — 魔導塔、結界門、聖域、魔法灯、発光する紋様や水晶、
  小さな浮遊要素など、幻想要素の現れやすさに影響する。
  最低値では魔法要素が消え、素朴な中世風の集落になる。
- **Monumentality / 中心建築の格** — 中心建築の大きさ、格、存在感、
  周辺の広場や道の集まり方に影響する。

## サマリーの読み方

生成が終わると、パネル下部に生成結果のサマリーが表示されます。

- **seed / パラメータ** — この箱庭を再現するための入力一式。
  既定値(50)と違うパラメータには印が付きます。
- **建物** — 生成された建物の総数と役割別の内訳
  (中心建築・住居・水辺・橋詰め・集会所・倉庫・外縁)。
- **中心建築の記述文** — 「水路の交わる魔法都市。三基の魔導塔と結界環を持つ。」
  のような、生成結果から機械的に作られる短い説明。
- **水辺** — 川の本数、湖、水路、橋の数。
- **結界** — 結界の段階(なし/境界標/部分結界/完全結界)、結界環の周長、
  魔導塔・結界門・聖域・浮遊要素の数。
- **スケール** — 箱庭の一辺の実寸と道路の総延長。
- **複雑度** — 描画負荷の目安(頂点数・インスタンス数・draw call)。
  端末に応じて表示品質は自動調整されますが、生成内容そのものは変わりません。
